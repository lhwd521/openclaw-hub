# 🚀 快速启动指南

## 立即开始使用

### 1. 启动服务器

```bash
cd openclaw-hub
npm run dev
```

浏览器会自动打开 `http://localhost:5173`

### 2. 登录

输入任意用户名，点击"进入"

### 3. 添加连接（至少 3 个）

点击左侧 "🔗 连接管理"，添加你的 OpenClaw 连接：

```
示例配置：

连接 1: 总经理
地址: https://manager.trycloudflare.com
密钥: your-token-1

连接 2: 数据分析师
地址: https://data-analyst.trycloudflare.com
密钥: your-token-2

连接 3: 财务顾问
地址: https://finance.trycloudflare.com
密钥: your-token-3
```

### 4. 构建组织架构

点击左侧 "🏢 组织架构"

**操作步骤：**

1. **拖拽"总经理"** 从左侧到右侧空白处 → 创建根节点
2. **拖拽"数据分析师"** 到"总经理"卡片上 → 成为下属
3. **拖拽"财务顾问"** 到"总经理"卡片上 → 成为下属
4. **点击 ✏️** 为每个节点设置职责：
   - 总经理 → "统一管理和协调团队"
   - 数据分析师 → "数据分析、网络爬虫、数据可视化"
   - 财务顾问 → "个人财务分析、投资建议、预算规划"
5. **点击"保存架构"**

### 5. 使用团队协作

点击左侧 "💬 聊天"

1. 选择"总经理"连接
2. 你会看到：
   ```
   总经理  ● 已连接  ☑ 🤖 团队协作
   👥 管理 2 个下属
   ```
3. **打开"🤖 团队协作"开关**
4. 输入测试任务：
   ```
   帮我分析特斯拉最近的股票走势，并给出投资建议
   ```
5. 观察总经理的回答

## 🎯 预期效果

### 系统自动做的事

当你发送消息时，系统会自动构建这样的完整消息发送给总经理：

```
[系统信息] 你是团队管理者，可以协调以下团队成员完成任务：

1. 数据分析师 (在线)
   职责: 数据分析、网络爬虫、数据可视化

2. 财务顾问 (在线)
   职责: 个人财务分析、投资建议、预算规划

如果需要团队协作，请返回 JSON 格式：
{
  "delegate": true,
  "tasks": [
    {"member": "成员名称", "task": "具体任务描述"}
  ],
  "reason": "分配理由"
}

如果你自己能完成，直接回答即可。

---

用户任务：帮我分析特斯拉最近的股票走势，并给出投资建议
```

### 总经理 AI 会

1. **看到团队信息** - 知道有哪些下属，每个下属的职责
2. **分析任务** - 判断是否需要团队协作
3. **做出决策**：
   - 选项 A：自己完成，直接回答
   - 选项 B：返回 JSON 任务分配方案

## 🐛 常见问题

### Q: 看不到"组织架构"导航项？

A: 检查是否已登录，刷新页面试试。

### Q: 拖拽不工作？

A: 确保使用的是现代浏览器（Chrome/Firefox/Edge），不支持 IE。

### Q: 聊天页面看不到团队协作开关？

A: 只有当前连接有下属时才会显示。检查组织架构是否正确设置。

### Q: 总经理没有收到团队信息？

A: 检查团队协作开关是否打开，检查浏览器控制台是否有错误。

### Q: 保存架构后刷新页面丢失了？

A: 检查浏览器是否允许 localStorage，不要使用无痕模式。
