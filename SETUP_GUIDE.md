# OpenClaw Hub - 智能团队协作系统使用指南

## 🎉 功能已完成！

现在你可以使用可视化的组织架构图来管理多个 OpenClaw，并实现智能任务分配。

## 快速开始

### 第一步：添加 OpenClaw 连接

1. 打开 OpenClaw Hub
2. 登录（输入用户名）
3. 点击左侧导航 "🔗 连接管理"
4. 添加你的多个 OpenClaw 连接

**示例：**
```
连接 1: 总经理 VPS
地址: https://manager.trycloudflare.com
密钥: abc123

连接 2: 数据分析 VPS
地址: https://data-analyst.trycloudflare.com
密钥: def456

连接 3: 财务顾问 VPS
地址: https://finance.trycloudflare.com
密钥: ghi789
```

### 第二步：构建组织架构

1. 点击左侧导航 "🏢 组织架构"
2. 从左侧"可用连接"区域，**拖拽**"总经理 VPS"到右侧空白画布
3. 继续拖拽"数据分析 VPS"到"总经理 VPS"卡片上（建立上下级关系）
4. 拖拽"财务顾问 VPS"到"总经理 VPS"卡片上
5. 点击每个节点的 ✏️ 按钮，设置职责描述：
   - 总经理 VPS → "统一管理和协调团队"
   - 数据分析 VPS → "数据分析、网络爬虫、数据可视化"
   - 财务顾问 VPS → "个人财务分析、投资建议、预算规划"
6. 点击右上角"保存架构"

### 第三步：使用团队协作

1. 点击左侧导航 "💬 聊天"
2. 选择"总经理 VPS"连接
3. 你会看到界面显示：
   ```
   总经理 VPS  ● 已连接  ☑ 🤖 团队协作
   👥 管理 2 个下属
   ```
4. 打开"🤖 团队协作"开关
5. 输入任务，例如："帮我分析特斯拉股票"
6. 系统会自动将团队信息发送给总经理 AI
7. 总经理 AI 会决定是否需要分配任务给下属

## 工作原理

### 当你向总经理发送消息时

**你输入：**
```
帮我分析特斯拉股票
```

**系统自动构建的完整消息：**
```
[系统信息] 你是团队管理者，可以协调以下团队成员完成任务：

1. 数据分析 VPS (在线)
   职责: 数据分析、网络爬虫、数据可视化

2. 财务顾问 VPS (在线)
   职责: 个人财务分析、投资建议、预算规划

如果需要团队协作，请返回 JSON 格式：
{
  "delegate": true,
  "tasks": [
    {"member": "成员名称", "task": "具体任务描述"}
  ],
  "reason": "分配理由"
}

如果你自己能完成，直接回答即可。

---

用户任务：帮我分析特斯拉股票
```

### 总经理 AI 的两种选择

**选项 1：自己完成**
- 直接回答用户问题
- 不返回 JSON

**选项 2：分配给团队**
- 返回 JSON 格式的任务分配方案
- 系统解析 JSON 并执行分配（未来版本实现）

## 使用技巧

### 1. 清晰的职责描述

职责描述会发送给上级 AI，帮助它决定任务分配。

**好的描述：**
```
数据分析 VPS
职责：
- 网络爬虫和数据抓取
- 数据清洗和预处理
- 数据可视化和报表生成
- 支持的数据源：股票、新闻、社交媒体
```

**不好的描述：**
```
数据分析 VPS
职责：做数据
```

### 2. 合理的层级设计

**推荐：** 2-3 层
```
总经理 → 部门经理 → 专员
```

**不推荐：** 过深的层级（效率低）

### 3. 专业化分工

让每个 OpenClaw 专注于特定领域，避免职责重叠。

### 4. 测试团队协作

先用简单的任务测试，例如：
```
"帮我查一下今天的天气，并给出穿衣建议"
```

看总经理 AI 是否能正确理解团队结构。

## 常见问题

### Q: 团队协作开关在哪里？

A: 只有当你选择的连接有下属时，聊天界面才会显示"🤖 团队协作"开关。

### Q: 如何知道总经理是否分配了任务？

A: 目前总经理的回答会直接显示在聊天中。如果它返回 JSON 格式，说明它想分配任务（但自动执行功能还未实现）。

### Q: 可以有多个总经理吗？

A: 可以！你可以创建多个独立的团队，每个团队有自己的领导。

### Q: 下属离线了怎么办？

A: 系统会在团队信息中标注"离线"，总经理 AI 会看到这个信息，通常会选择其他在线的下属。

### Q: 如何删除组织架构？

A: 点击组织架构页面右上角的"重置"按钮。

### Q: 职责描述可以修改吗？

A: 可以！点击节点上的 ✏️ 按钮随时修改。
