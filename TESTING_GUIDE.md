# 测试指南

## 快速测试步骤

### 1. 启动开发服务器

```bash
cd openclaw-hub
npm install  # 如果还没安装依赖
npm run dev
```

### 2. 测试 PDF 文件上传

1. 打开浏览器访问 `http://localhost:5173`
2. 登录并连接到一个 VPS
3. 点击聊天输入框左侧的附件图标
4. 选择一个 PDF 文件
5. 等待提取完成（会显示"正在提取文件内容..."）
6. 预览区应显示 "PDF" 图标和文件名
7. 点击发送
8. 消息应包含提取的 PDF 文本内容

### 3. 测试 Word 文档上传

1. 选择一个 .docx 文件
2. 等待提取完成
3. 预览区应显示 "DOCX" 图标和文件名
4. 发送后消息应包含文档文本

### 4. 测试 Excel 表格上传

1. 选择一个 .xlsx 或 .xls 文件
2. 等待提取完成
3. 预览区应显示 "XLSX" 或 "XLS" 图标
4. 发送后消息应包含 Markdown 格式的表格

### 5. 测试图片上传

1. 选择一个图片文件（.jpg, .png, .gif, .webp）
2. 预览区应显示图片缩略图
3. 发送后消息应显示压缩后的图片

### 6. 测试多文件上传

1. 同时选择多个文件（最多 4 个）
2. 预览区应显示所有文件
3. 可以单独删除某个文件
4. 发送后所有文件内容应正确处理

### 7. 测试错误处理

1. 尝试上传超过 10MB 的文件 → 应显示错误提示
2. 尝试上传不支持的文件类型 → 应显示错误提示
3. 尝试上传超过 4 个文件 → 应显示限制提示

## 预期结果

### PDF 文件
- 提取所有页面的文本内容
- 文本按页面顺序排列
- 保留基本的段落结构

### Word 文档
- 提取所有文本内容
- 保留基本的段落结构
- 不包含图片和格式

### Excel 表格
- 每个工作表转换为独立的 Markdown 表格
- 表格包含表头和数据行
- 工作表名称作为二级标题

### 图片
- 压缩到最大 1024x1024 像素
- JPEG 质量 0.7
- 保持宽高比
- 文件大小通常在 200KB 以下

## 调试技巧

### 1. 查看控制台日志
打开浏览器开发者工具（F12），查看 Console 标签页：
- 文件提取错误会显示在控制台
- 网络请求失败会显示在 Network 标签页

### 2. 检查网络请求
在 Network 标签页中：
- 查看 CDN 库的加载情况
- 检查 WebSocket 消息大小
- 确认文件内容是否正确发送

### 3. 测试 CDN 连接
在控制台中手动测试 CDN 加载：
```javascript
// 测试 PDF.js
import('https://cdn.jsdelivr.net/npm/pdfjs-dist@4.9.155/+esm').then(console.log)

// 测试 Mammoth.js
import('https://cdn.jsdelivr.net/npm/mammoth@1.8.0/+esm').then(console.log)

// 测试 SheetJS
import('https://cdn.jsdelivr.net/npm/xlsx@0.18.5/+esm').then(console.log)
```

## 常见问题

### Q: 文件提取很慢
A: 首次加载需要从 CDN 下载库，后续会使用浏览器缓存。大文件提取也需要更多时间。

### Q: PDF 提取失败
A: 确保 PDF 不是扫描版（纯图片），扫描版 PDF 需要 OCR 才能提取文字。

### Q: Excel 表格格式混乱
A: 复杂的 Excel 格式（合并单元格、公式等）可能无法完美转换，建议使用简单的表格结构。

### Q: 图片上传后看不到
A: 检查图片格式是否支持，确认压缩过程没有错误。

### Q: CDN 加载失败
A: 检查网络连接，确认可以访问 cdn.jsdelivr.net。可以考虑使用其他 CDN 或本地部署库文件。

## 性能测试

### 建议测试文件大小
- PDF: 1-5MB，10-50 页
- Word: 100KB-2MB，10-100 页
- Excel: 100KB-5MB，1-10 个工作表
- 图片: 500KB-5MB

### 预期处理时间
- PDF (10 页): 2-5 秒
- Word (50 页): 1-3 秒
- Excel (5 个工作表): 2-4 秒
- 图片: 0.5-1 秒

## 浏览器兼容性

测试通过的浏览器：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

不支持的浏览器：
- IE 11 及更早版本
- 旧版移动浏览器
